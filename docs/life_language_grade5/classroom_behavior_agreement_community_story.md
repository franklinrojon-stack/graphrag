# 课堂行为支持小协议｜《社区故事采写》

## 微技能目标
- 专注倾听、轮流发言、尊重受访者。

## 协议内容
1. **准备姿态**：采访前深呼吸一次，双手放于桌面，目视对方。
2. **提问礼仪**：用“请问……可以分享吗？”开头，等待3秒再追问。
3. **记录礼仪**：记录时不打断，用点头示意“我在听”。
4. **反馈语言**：分享时先说“谢谢你愿意告诉我……”，再提出建议。

## 视觉提示
- 桌面放置“倾听三步卡”：眼神→点头→记录。
- 使用红绿夹子提醒“准备好/稍等”。

## 强化方式
- 小组达成协议可获得“社区记者徽章”。
- 记录一次积极行为贴纸，可兑换展示机会。

## 危机预防
- 若学生情绪紧张，提供“安静角”2分钟调整，使用情绪卡选择状态。
