# 评估与基线：游戏互动语言样本记录

## 学生概况
- 年龄：5岁6个月，ASD+ID，接受早期干预第6个月
- 当前沟通方式：自然发声（单词、音节）、手势指点、PECS 一级交换
- 主要沟通意图覆盖：请求、抗拒；缺乏评论、社交开启

## 基线总结
- 现有沟通方式以手势和单词为主，缺少多词短句。
- 评论和社交开启意图接近零，需通过自然情境引入。
- 建议优先目标：请求帮助、评论、轮换提示，结合 AAC 模型提升清晰度。
