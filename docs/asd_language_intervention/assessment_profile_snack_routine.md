# 评估与基线：点心时间沟通功能扫描

## 学生概况
- 年龄：7岁2个月，ASD+中度智能障碍，融合班随班就读
- 当前沟通方式：语音输出设备（GoTalk 20+）、手势“停”、仿说短句
- 主要沟通意图覆盖：请求食物、表达拒绝；缺乏求助、评论同伴

## 口肌与言语可懂度
- 口肌检查：唇闭合力量弱，咀嚼偏向单侧；建议每节课加入口肌热身
- 可懂度：在熟悉主题时短句可懂度约60%，陌生情境降至30%
- 语速：短句平均1.8秒，存在语调单一问题

## 沟通功能覆盖
- 请求：频次高但多为非口语动作，需转换到 AAC 输出
- 求助：缺乏，面对困难任务以尖叫表达
- 社交开启：缺乏，需在同伴互动中设置脚本
- 评论：未观察到

## 基线总结
- 现有 AAC 使用多为被动响应，缺少主动发起
- 需将求助、轮替提示纳入初期训练目标，并结合口肌练习提升发音稳定性
- 建议设置家庭点心场景泛化任务，记录独立率
