## 进展判定规则
- 独立率连续两周≥80%，反应时长≤4秒且设备故障=0，即可减少成人监控。
- 若反应时长>6秒，增加延迟提示训练与设备熟悉回合。
- 每周向家长分享数据图表，并协调家庭场景训练。
